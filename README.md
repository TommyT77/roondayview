Local Machine:

Npm i (For both frontend and backend)

    Bcrypt
    Cors
    Dotenv
    Express
    Helmet
    Jsonwebtoken
    Mongoose
    Jest
    Supertest
    nodemon

    emotion/react
    emotion/styled
    mui/icons-material

    mui/material

    autosuggest-highlight
    styled-components
